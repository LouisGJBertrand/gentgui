How it works?

This API is based on 'class' tag but still compatible with other tools like bootstrap."but how is it compatible?" you might say. well it uses group classes, that mean that it formed like it will not affect other APIs, let me explain;

You use several class tags like "gentgui_button_discord_Blurple" or "gentgui_form_container". you can figure that each gentgui's classes has the same construction That's begins with "gentgui_" then followed by the "element_" parents and then the actual element and it ends with some extra things like "big" or "warn".

here is a little graphic explanation of multiple class!




ELEMENTS ShowDown

big buttons
gentgui_button_big
Class names:

gentgui_button_big => button base style

gentgui_button_color => color of the button

e.g.
<button class="gentgui_button_light gentgui_button_big">button.light</button>


little buttons
gentgui_button_lit
Class names:

gentgui_button_lit => button base style

gentgui_button_color => color of the button

e.g.
<button class="gentgui_button_softwarn gentgui_button_lit">button.light</button>


Forms
gentgui_form
Class names:

gentgui_form => form style

gentgui_form_input_textbox => Text box style

gentgui_form_input_pass => add special effects for passwords

gentgui_form_input_sub => just the button

e.g.
<form method="post" class="gentgui_form gentgui_form_centered">

	<div class="gentgui_form_container">

		<h1>CONNECTION SAMPLE</h1>

		login infos : <input type="text" name="login" class="gentgui_form_input_textbox"><br>
		password : <input type="password" name="pass" class="gentgui_form_input_textbox gentgui_form_input_pass"><br>
		<input type="submit" class="gentgui_button_softwarn gentgui_button_lit gentgui_form_input_sub"><br>
	
	</div>

</form>
